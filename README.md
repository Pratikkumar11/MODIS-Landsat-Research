# MODIS–Landsat Thermal Image Fusion using Deep Learning

## Overview

This repository presents a comprehensive research study on satellite image fusion using MODIS and Landsat datasets with Deep Learning techniques.

The project explores how low-resolution MODIS imagery can be combined with high-resolution Landsat imagery to generate enhanced thermal images suitable for environmental monitoring, agriculture, climate analysis, and disaster management.

This repository documents the complete research process, including literature review, remote sensing fundamentals, dataset analysis, preprocessing techniques, image fusion methodologies, deep learning implementation, experimental observations, and future research directions.

---

## Research Objectives

- Study Remote Sensing fundamentals
- Understand MODIS and Landsat satellite systems
- Review image fusion techniques
- Explore Deep Learning approaches for satellite imagery
- Analyze preprocessing methods
- Develop a CNN-based image fusion model
- Document research findings and future scope
